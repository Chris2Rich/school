Proof By Induction


Concept: By proving true for a set (usually 1) and then assuming true for a similar set k, if it is possible to get to set k+1 then you can use inductive reasoning to see that this is true for a larger set (usually Z+).


Prove by mathematical induction that, for n∈N


∑r=1n1(2r-1)(2r+1)=n2n+1

Prove true for 1:


∑r=111(2r-1)(2r+1)=13=1(2×1)+1=13

Assume true for n=k


∑r=1k1(2r-1)(2r+1)=k(2k)+1

Therefore:


∑r=1k+11(2r-1)(2r+1)=k(2k)+1+1(2k+1)(2k+3)

Which can be rewritten as:


∑r=1k+11(2r-1)(2r+1)=k+12(k+1)+1

As true for n=1 and assuming true for n=k, this is true for n=k+1, this generalises to all n∈N.
Obviously you must show intermediate steps of correct algebra.
